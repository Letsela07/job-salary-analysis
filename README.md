# Job Salary Analysis Project

## Overview
End-to-end data analysis project using MySQL and Looker Studio
analyzing 250,000 job salary records from Kaggle.

## Stages
- Stage 1: Data Loading — MySQL database setup
- Stage 2: Data Cleaning — NULL checks, duplicates, consistency
- Stage 3: EDA — SQL queries to find salary insights
- Stage 4: Dashboard — Looker Studio visualizations

## Key Findings
- Location is the biggest salary driver (84k gap USA vs India)
- AI Engineer is the highest paying role (173k avg)
- Enterprise companies pay 42k more than Startups
- Education has moderate impact — PhD earns 32k more than High School

## Tools Used
- MySQL
- VS Code
- Looker Studio
- GitHub

## Dataset
- Source: Kaggle
- 250,000 rows
- 10 columns
